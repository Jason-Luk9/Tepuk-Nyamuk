# 🦟 Tepuk Nyamuk

A native iOS reaction card game built for the **WWDC 2026 Swift Student Challenge**, 
inspired by a traditional Indonesian card-slapping game.

## About
Tepuk Nyamuk is a fast-paced reaction game where you compete against a CPU opponent. 
Cards are flipped onto a center pile while a count cycles from Ace to King — 
when the spoken count matches the top card, slap it before the CPU does!

Built to preserve a piece of Indonesian childhood culture and bring it to a global audience.

## Features
- Real-time card battle against CPU opponent
- CPU AI with difficulty-scaled reaction timing (Easy / Medium / Hard)
- Full audio with background music, sound effects, and haptic feedback
- Accessibility-first — spoken count always displayed visually
- Retro pixel art aesthetic with 8-bit fonts and chiptune audio
- Settings for music, SFX, and vibration

## Tech Stack
- Swift & SwiftUI
- MVVM Architecture
- AVFoundation

## Requirements
- Xcode 26 or later
- iOS 17+

## How to Run
Open `Tepuk Nyamuk.swiftpm` in Xcode and run on simulator or device.
